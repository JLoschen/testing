Super Duper Jumper

Tech Used:Android(Java, Android Play Store process), Game Dev(sprites, game physices, game loop) This project taught me to KISS.

I only had 2 years of CS under my belt at the time and this was the first thing I programmed because I wanted to and not for an assignment. Neighborhood Map

Tech Used:Knockout JS, Fetch API, Google Maps API, 4square API

Summary:This was a pretty cool project for Udacity where we were supposed to provide a map of places in our neighborhood as well as use a 3rd party API. The map allows the user to filter out locations on the left. It was also the first time I had used a JS framework instead of just writing vanilla Javascript. Coming from a WPF background I was all about using DataBinding and separation of concerns in this project.

Rearranging Grid

Tech Used: C#, WPF, XAML

Summary:This User Control extends the basic WPF Grid by allowing a min width or min height where anything smaller will cause the Grid to rearrange or hide some of it's rows or columns. It's basically useful for doing the WPF version of Responsive design. If the window shrinks certain rows or columns containing less important elements can be hidden. Also if the aspect ratio becomes too skinny or wide the contents can rearrange.
